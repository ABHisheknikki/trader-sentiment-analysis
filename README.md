
# Trader Behavior vs Market Sentiment Analysis

**Author:** Abhishek Pratap Singh
**Role:** Data Science Intern

---

## Overview
This project analyzes how trader profitability and activity vary with overall Bitcoin market sentiment using real trade data and the Fear & Greed Index.
The analysis identifies behavioral trends — such as how traders perform during *fearful* vs *greedy* markets — and visualizes insights through data-driven storytelling.

---

## Objectives
- Clean and preprocess 200K+ crypto trade records
- Merge trading data with the daily Fear & Greed Index
- Analyze relationships between market sentiment, profitability, and trading activity
- Visualize key findings using Seaborn and Matplotlib
- Document insights and behavioral patterns in a structured report

---

## Tech Stack
- **Language:** Python
- **Libraries:** pandas, numpy, matplotlib, seaborn
- **Tools:** Google Colab, GitHub, Google Drive
- **Output:** Report (`ds_report.pdf`) and visualizations (`/outputs/` folder)
